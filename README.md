## Scientific Research Project Acceptance Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/6a8cb15f0bed4d1f8886d8cf28ab351f/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, project administrators, expert accounts, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Change Password
4. Personal Center
5. Announcement Information
6. Project Management



**Project Administrator Features:**
1. Login
2. Change Password
3. Personal Information
4. Announcement Information
5. Project Management (Project Review)



**Expert Account Features:**
1. Login
2. Change Password
3. Personal Information
4. Announcement Information
5. Project Information





**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Basic Data Management (Announcement Type Management, Project Type Management)
4. Announcement Information Management
5. Review Log Management
6. Project Management
7. Project Administrator Management
8. User Management
9. Expert Account Management
